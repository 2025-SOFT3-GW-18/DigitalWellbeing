# Digital Wellbeing （スマホ依存対策Webサイト）

ユーザーの「スマホ依存度」を診断し、その傾向に合った対策を見つけ、知識を深めるためのデジタル・ウェルビーイング支援サイトです。

## 📱 サイトの概要

簡単な質問に答えるだけで、スマホへの依存度を客観的に数値化し、タイプ別の対策を提案します。
Next.jsによるSPA（シングルページアプリケーション）として動作し、診断回答・履歴・投票などのユーザーデータはサーバーへ送信せずブラウザ内（LocalStorage）に保存されます。

## ✨ 主な機能

### 1. 🩺 依存度診断 (Diagnosis)
* **スマホ依存度チェック**: 10個の質問で依存度をスコア化（30点満点）。
* **レベル判定**: 「低依存」～「重度依存」の4段階で判定し、レベルに応じたアドバイスを表示。
* **履歴グラフ**: 過去の診断結果をグラフで可視化し、前回スコアとの比較を表示して改善意識を高めます。

### 2. 🔍 タイプ診断 (Personalize)
* **依存タイプ分析**: 3つの質問で、依存の傾向を4タイプに分類。
    * 🐰 SNS・承認欲求タイプ
    * 🎮 ゲーム・没頭タイプ
    * 👻 無意識・習慣タイプ
    * 💼 仕事・強迫観念タイプ
* **パーソナライズ提案**: 診断されたタイプに合わせて、最適な対策ツール（アプリ）とアナログ趣味を自動で提案します。

### 3. 📚 ガイド & 評価 (Resources)
* **対策ツール一覧**: 集中力育成、強制ロック、管理系などカテゴリ別に紹介。
* **多角的な評価システム**: 各アプリに対し「効果・楽しさ・手軽さ・継続性・デザイン」の5項目で投票が可能。
* **データ可視化**: ユーザーの投票結果を集計し、レーダーチャートで特徴をリアルタイムに表示。

### 4. 🎨 趣味（アナログ置き換え）(Hobby)
* **趣味カタログ**: デジタルデトックスやリラックスに向いた趣味をタイプ別に紹介。
* **タイプ別おすすめ**: タイプを選択すると、そのタイプに合ったおすすめ趣味が表示されます（診断済みの場合は結果を表示）。
* **絞り込み**: 難易度（初級/中級/上級）、初期費用（無料/低/中/高）でフィルタ可能。
* **詳細表示**: 「最初の一歩」「準備物」「屋内/屋外」などをモーダルで確認できます。

### 5. 🧠 知識・相談 (Knowledge)
* **学習リソース**: 脳科学的な依存の仕組みやデジタルデトックスに関する情報リンク。
* **相談窓口**: 都道府県や専門機関の相談窓口への案内。

### 6. 👤 ユーザー管理・管理者機能
* **ユーザープロフィール**: ログイン、新規登録、アイコン変更などの擬似的な認証機能（ローカル保存）。
* **管理者モード**: 管理者権限でログインすることで、デモデータの自動生成や全ユーザーデータの初期化などのメンテナンス操作が可能。

## 🛠️ 使用技術

* **Framework**: Next.js 14+ (App Router / Client Component)
* **Language**: TypeScript / React
* **Styling**: Tailwind CSS
* **Charts**: Chart.js（CDNから動的ロード）
* **Storage**: LocalStorage (ブラウザ保存)

## 🚀 始め方 (Getting Started)

このアプリを動かすには **Node.js** が必要です。
ターミナル（PowerShellやコマンドプロンプト）で以下のコマンドを入力して確認してください。

```bash
node -v
```

PCにインストールしていない場合は、以下の記事を参考に準備してください。

* [【Windows】Nodejsをインストールしよう](https://zenn.dev/kuuki/articles/windows-nodejs-install)

プロジェクトのディレクトリで以下のコマンドを順に実行し、インストールと起動を行います。

```bash
npm install
```
```bash
npm run dev
```

コマンド実行後、ブラウザで http://localhost:3000 を開くとサイトが表示されます。

## 🔐 パスワードポリシー

本サイトでは、アカウント登録・プロフィール更新時に以下のパスワード基準を適用します。

### 許可文字と長さ
- **許可文字**：半角のASCII印字可能文字（`!` ～ `~` ／英数字・記号）
- **文字数**：8～64文字
  - 8文字未満／65文字以上は不可。
  - 入力中は全角・空白などの不正文字を自動除去し、最大64文字に切り詰めます。

### 禁止パターン（弱パターン）
次のいずれかを含むパスワードは登録・更新できません。
1. **同一文字の4連続**  
   例：`aaaa` / `1111` / `!!!!`
2. **連番・キーボード配列の単純な並び**（**正順・逆順とも不可**）  
   例：`1234` / `abcd` / `qwerty` / `lkjhg` / `9876` / `dcba`
3. **一般的に弱い単語の含有**（**大文字小文字を区別せず**）  
   例：`password` / `admin` / `qwerty` / `abc123` / `111111` / `123456` / `123456789`
